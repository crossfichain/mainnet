### Initialize Chain

```bash
wget https://github.com/mineplexio/mineplex-2-node/releases/download/v0.1.0/mineplex-2-node._v0.1.0_linux_amd64.tar.gz && tar -xf mineplex-2-node._v0.1.0_linux_amd64.tar.gz
git clone https://github.com/mineplexio/mainnet.git
./mineplex-chaind start --home ./mainnet
```
